# Changelog

## [1.0.0] - 2026-02-07

Initial public release.

### Features

- **Voice calling** — Real-time phone conversations with your Clawdbot
- **SMS messaging** — Send and receive text messages
- **Missions** — Dispatch AI assistants to make calls or send SMS
- **WebSocket client** — Connects your gateway to ClawdTalk
- **Setup wizard** — Auto-configures gateway and generates config
